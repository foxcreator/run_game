# 🎮 Новий Екран Завантаження з Порадами

## ✨ Що додано:

### 1. **Поради для гравців** (15 штук)
Всі поради в конфігу `GAME_CONFIG.LOADING_TIPS`:
- 💡 Використовуй самокат для швидкого втечі від ТЦК
- 🚗 Машини можуть збити не тільки тебе, а й працівників ТЦК
- 💰 Чим довше граєш без банку, тим більший множник ризику
- ... та ще 12 порад!

### 2. **Красивий UI в стилі гри**
- 🎨 Золотий прогрес бар з обводкою
- 📊 Великий заголовок "BUSIFICATION"
- 💬 Поради в рамці з синьою обводкою
- ⏱️ Мінімум 3 секунди показу (щоб встигнути прочитати)

### 3. **Рандомні поради**
- 🔄 Змінюються кожні 5 секунд
- 🎲 Завжди різні (не повторюються підряд)
- 📱 Працюють на всіх екранах завантаження

---

## 📁 Структура:

### `src/config/gameConfig.js`
```javascript
LOADING_TIPS: [
    '💡 Використовуй самокат для швидкого втечі від ТЦК',
    '🚗 Машини можуть збити не тільки тебе, а й працівників ТЦК',
    // ... додавай нові поради тут!
]
```

### `src/utils/LoadingScreen.js`
Універсальний клас для екранів завантаження:
- Створює UI
- Показує рандомні поради
- Контролює мінімальний час показу (3 сек)
- Автоматично змінює поради кожні 5 сек

### Використовується в:
- `BootScene` - перше завантаження гри
- `GameScene` - завантаження при вході в гру

## ⚙️ Налаштування:

### Мінімальний час показу:
```javascript
// src/utils/LoadingScreen.js, рядок 10
this.minLoadingTime = 3000; // 3 секунди
```

### Інтервал зміни порад:
```javascript
// src/utils/LoadingScreen.js, рядок 136
delay: 5000, // 5 секунд
```

---

## 📝 Як додати нову пораду:

1. Відкрий `src/config/gameConfig.js`
2. Знайди `LOADING_TIPS`
3. Додай новий рядок:
```javascript
LOADING_TIPS: [
    '💡 Використовуй самокат...',
    '🚗 Машини можуть збити...',
    '🆕 Твоя нова порада тут!', // ← Додай сюди
]
```
4. Готово! Порада автоматично з'явиться в пулі

---

## 🎯 Особливості:

✅ **Мінімум 3 секунди** - навіть якщо все завантажилось миттєво
✅ **Рандомні поради** - кожен раз нова
✅ **Зміна кожні 5 сек** - якщо завантаження довге
✅ **Не повторюються** - нова порада завжди інша
✅ **Легко додавати** - просто дописуй в масив
✅ **Красивий UI** - в стилі всієї гри

---

## 🎮 Тестування:

1. Перезавантаж гру
2. Побачиш новий екран завантаження з порадою
3. Якщо завантаження довге - поради змінюватимуться кожні 5 сек
4. Мінімум 3 секунди показу (навіть якщо все готово)
